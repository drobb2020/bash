# Bash Scripting Repository

This is my bash scripting repository that I use to help manage customers environment for my day to day job. These scripts have all been tested and work on either SLES10 SPx, SLES11 SPx and OES2, OES11, OES2015. There are only a few scripts that will work on SLES12 SPx, and OES2018.
